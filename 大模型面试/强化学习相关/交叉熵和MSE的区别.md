# 交叉熵和MSE的区别

交叉熵（Cross-Entropy）和均方误差（Mean Squared Error, MSE）是机器学习和深度学习中最常用的两种损失函数。它们的核心区别在于**适用的任务类型**以及**在反向传播时的梯度表现**。

## 1. 核心应用场景不同
*   **MSE（均方误差）**：主要用于**回归任务（Regression）**。即预测连续的数值，例如预测房价、温度、股票价格等。
*   **交叉熵（Cross-Entropy）**：主要用于**分类任务（Classification）**。即预测离散的类别概率，例如判断图片是猫还是狗（二分类），或者手写数字识别（多分类）。

## 2. 数学公式与物理意义
*   **MSE**：衡量的是预测值与真实值之间的**欧式距离**（差异的平方）。
    $$MSE = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$
    *其中 $y_i$ 是真实值，$\hat{y}_i$ 是预测值。*

*   **交叉熵**：衡量的是两个**概率分布**之间的差异。真实标签通常被表示为 One-Hot 编码（如 `[0, 1, 0]`），预测值是经过 Softmax 或 Sigmoid 输出的概率分布（如 `[0.1, 0.8, 0.1]`）。
    $$CE = - \sum_{i=1}^{n} y_i \log(\hat{y}_i)$$
    *其中 $y_i$ 是真实标签的概率（通常是 0 或 1），$\hat{y}_i$ 是模型预测的概率。*

## 3. 为什么分类任务不用 MSE 而用交叉熵？（核心重点）
这是面试中经常被问到的问题，主要原因在于**梯度消失问题**和**收敛速度**。

在分类任务中，模型的最后一层通常会加上 **Sigmoid**（二分类）或 **Softmax**（多分类）激活函数，将输出映射到 $(0, 1)$ 之间。

*   **如果使用 MSE + Sigmoid**：
    根据链式法则，损失函数对权重的偏导数中会包含 Sigmoid 函数的导数 $\sigma'(x) = \sigma(x)(1-\sigma(x))$。
    当模型预测极其错误时（例如真实标签是 1，但模型预测接近 0），Sigmoid 的导数 $\sigma'(x)$ 会非常接近于 0。这会导致**梯度消失**，参数几乎不更新，模型“错得越离谱，学得越慢”。
*   **如果使用 交叉熵 + Sigmoid/Softmax**：
    交叉熵中的对数函数 $\log$ 巧妙地抵消了 Sigmoid/Softmax 导数中的复杂部分。最终计算出的梯度直接正比于**预测值与真实值的差**：$Gradient \propto (\hat{y} - y)$。
    这意味着：**模型预测错得越离谱，误差 $(\hat{y} - y)$ 越大，梯度就越大，模型参数更新得就越快**。这非常符合人类的学习直觉。

## 4. 对异常值（Outliers）的敏感度
*   **MSE**：由于计算的是误差的平方，如果存在偏离极大的异常值，误差会被平方放大，导致模型过度关注这些异常值，从而影响整体性能。
*   **交叉熵**：只关注真实类别对应的预测概率的对数，对其他类别的预测值不那么敏感，因此在分类问题中表现更稳定。

## 总结对比表

| 特性 | MSE (均方误差) | Cross-Entropy (交叉熵) |
| :--- | :--- | :--- |
| **主要任务** | 回归 (Regression) | 分类 (Classification) |
| **输出层激活函数** | 通常无 (Linear) | Sigmoid (二分类) / Softmax (多分类) |
| **物理意义** | 预测值与真实值的几何距离 | 预测概率分布与真实概率分布的差异 |
| **梯度表现(配合Sigmoid)**| 易发生梯度消失，错得越离谱学得越慢 | 梯度线性依赖于误差，错得越离谱学得越快 |
| **公式** | $\frac{1}{n} \sum (y - \hat{y})^2$ | $- \sum y \log(\hat{y})$ |
